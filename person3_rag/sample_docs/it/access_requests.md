# IT Access Request Procedure

## Standard Access
New employees receive standard access (email, intranet, VPN) automatically on
their start date. No request is needed.

## Privileged Access
Access to production systems, admin consoles, or customer data requires an
access request approved by the system owner and the IT security team. Specify
the system, the justification, and the duration (max 90 days, renewable).

## Security Guidelines
All privileged access is logged and reviewed quarterly. Multi-factor
authentication is mandatory for any access to production environments.
